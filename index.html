CSS Visual Rules

Welcome! Please refer to the following documentation as a basic "cheat sheet" as you build CSS stylesheets.

Though not an exhaustive resource, this should get you up and running in no time. This resource assumes you have a basic understanding of the HTML language and structure.

Formatting Rulesets

Before going any further, here is a reminder that I have to give myself every time I start a stylesheet: Don't forget to link your CSS stylesheet to your HTML document! Always include the following line of code within the <head> of your HTML file:

<link href="nameOfStyleSheet.css" rel="stylesheet">

A ruleset defines the style that applies to different elements in an HTML file. Within the CSS stylesheet, rulesets are formatted as follows:

selector {

property: value;

}

The selector indicates what element(s) a CSS style applies to. It is important to pay attention to the specificity of your selector. Rulesets with the highest specificity will be applied to their corresponding HTML element(s) instead of rulesets with less specificity.


Selectors

The following table summarizes the CSS syntax and specificity for id, class, attribute, or type HTML elements. A universal selector applies styling to all elements in an HTML file; it has no specificity.

Element HTML Format, Example CSS Selector Format, Example Specificity
ID <type id='id'>, <p id='apple'> #id {}, #apple {} 100
Class <type class='class'>, <p class='banana'> .class {}, .banana {} 10
Attribute <attribute>, <strong> attribute {}, strong {} 10
Type <type>, <p> type {}, p {} 1
Universal * {}, * {} 0

Properties and Values

The following table includes many of the CSS properties that define the visual style of HTML elements as well as their corresponding values and any important considerations when using each property.

Property Syntax Example Values Purpose Important Notes
Font family font-family: value; 'Garamond', 'Times New Roman', Montserrat, serif Changes the font style of an element Multi-word font families like Times New Roman require quotes around them, 'Times New Roman'.
Font size font-size: value; 18px, 10px Changes the font size of an element
Font weight font-weight: value; bold, normal Changes the weight of an element's font The value for the font-weight can also be a number between 0 and 900; bold corresponds to 900.
Text align text-align: value; left, center, right, justify Aligns text within its parent element
Font color color: value; blue, rgb(), rgba(), hsl(), hsla() Changes the font color within an element
Background color background-color: value; blue, rgb(), rgba(), hsl(), hsla() Changes the color of an element's background
Opacity opacity: value; 0, 0.3, 0.8, 1 The measure of how transparent an element is The opacity value can be any decimal number between 0 and 1.
